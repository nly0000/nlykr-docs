# nly.kr 문서

이 저장소는 nly.kr 서비스에 대한 개요와 문서를 제공합니다.

nly.kr는 URL 단축과 온라인 문서(PDF) 생성을 제공하는 웹 기반 플랫폼입니다.

공식 웹사이트: https://nly.kr

---

## 개요

nly.kr는 링크 공유와 문서 배포를 단순화하는 것을 목표로 하는 경량 웹 서비스입니다.

별도의 회원가입 없이도 기본 기능을 사용할 수 있으며,  
회원 가입 시 추가적인 관리 및 통계 기능이 제공됩니다.

---

## URL 단축 서비스

URL 단축 기능은 긴 주소를 짧고 공유하기 쉬운 링크로 변환합니다.

### 주요 특징
- 로그인 없이 즉시 URL 단축 가능
- 짧고 단순한 URL 구조
- HTTPS 기반 안전한 리다이렉션
- QR 코드 자동 생성
- 사용자 지정 주소(커스텀 슬러그) 지원
- 회원 대상 클릭 및 접근 통계 제공
- 개발자용 API 제공

### 활용 사례
- 메신저 및 SNS 링크 공유
- 오프라인 인쇄물용 QR 코드 생성
- 캠페인 및 마케팅 링크
- 임시 또는 일회성 링크 생성

자세한 내용은 아래 페이지를 참고하세요.  
https://nly.kr/product

---

## PDF 문서 생성 서비스

nly.kr는 웹에서 바로 문서를 작성하고 PDF로 생성할 수 있는 기능을 제공합니다.

견적서, 이력서, 회의록, 재직증명서 등의 문서를 작성한 후  
PDF 다운로드 또는 링크 형태로 공유할 수 있습니다.

### 주요 기능
- 웹 기반 문서 작성
- PDF 파일 생성 및 링크 공유
- 비공개 문서 및 비밀번호 보호
- 문서 조회 통계 제공
- 업무 및 행정 문서 활용에 적합

관련 서비스 페이지:  
https://nly.kr/pdf-product-new

## API 제공

nly.kr는 URL 단축 기능을 외부 서비스에 연동할 수 있도록 API를 제공합니다.

API 사용 방법과 정책은 공식 웹사이트에서 확인할 수 있습니다.
